# mm-backend-core

CLI to generate the mm-backend-core backend template.

## Usage

Global install:

```sh
npm i -g mm-backend-core
mm-backend-core init my-app
```

Local install + npx:

```sh
npm i mm-backend-core
npx mm-backend-core init my-app
```

## Quick Start

```sh
npm i -g mm-backend-core
mm-backend-core init my-app
cd my-app
npm install
npm run dev
```
